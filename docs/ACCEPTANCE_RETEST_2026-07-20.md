## 本轮发现并修复的代码缺陷

1. 新闻回答无图：视觉 Provider 缺密钥时把 SearchPro 已返回的文章主图也丢弃。现仅在模型明确判定不相关时丢图，其余失败保留 HTTPS 主图。
2. 图片版本刷新回退：历史消息恢复只读旧快照。现从 Makers Workspace 重建当前 Action 与版本链。
3. 大 PDF 502：Cloud Functions 响应上限 6MB。现用 4MB 鉴权分片从 Makers Blob 读取，前端合并。
4. 中文文件名 502：原始 CJK 字符写入 `Content-Disposition` 触发 Response ByteString 异常。现使用 ASCII fallback 与 RFC 5987 `filename*`。
5. arXiv 下载 504：平台默认 30 秒不足。现通过 `edgeone.json` 复用 Makers 可配置的 120 秒时限，并在 105 秒由业务返回友好错误。
