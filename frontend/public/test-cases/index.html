怎么使用

  1. 先选择测试环境并填写部署 ID；不要在生产环境试验破坏性或付费场景。
  2. 按表格中的“前置条件 → 测试数据 → 操作步骤 → 预期结果”逐条执行。
  3. 每条用例选择结果并填写证据/问题。记录只保存在当前浏览器。
  4. 点击“导出验收记录”保存 JSON；生产阻断用例全部通过后再申请生产发布。
