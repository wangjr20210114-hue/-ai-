怎么使用

  1. 先选择测试环境并填写部署 ID；不要在生产环境试验破坏性或付费场景。
  2. 打开一个 Case 后，从“步骤 1”向下执行；每一步都在同一行写明操作位置、按钮文字、输入内容和本步预期。
  3. 为当前电脑填写“主机名称”。结果、备注、图片/视频证据会同步到 Makers Blob，多台电脑打开同一站点即可共享。
  4. “测试数据”右侧可直接点复制。不要自己改写测试文案，否则不同测试人的结果无法比较。
  5. 备注只填写文字现象；截图和录屏从“证据”入口上传。每次编辑都会记录主机、测试人和时间。
  6. 生产阻断用例全部通过前禁止发布生产；JSON 导出仅作为离线备份。
